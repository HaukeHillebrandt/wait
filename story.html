# Mine a block = find a short vector in a lattice rebuilt every block.
#   A    = SHAKE256(parent_hash || nonce)   ->  Z_q^{n x m}
#   win  = v != 0  with  A·v ≡ 0 (mod q)  and  ‖v‖ ≤ β
#
# Pin β just above the shortest-vector length λ₁:
#   loose β  ->  many solutions, lattice reduction cracks it free  (no work)
#   tight β  ->  search cost 2^Θ(blocksize)                        (the difficulty)
def verify(A, v, q, beta):
    return any(v) and (A.dot(v) % q == 0).all() and norm(v) <= beta   # one matvec

In plain terms: mining is finding a needle in a lattice — brutal to find, instant to check. Size the haystack so it chokes on memory bandwidth, not raw speed, and a hobbyist's PC keeps pace with a server farm. Money I print by thinking, that nobody can counterfeit and no datacenter can corner.

One part has to be a theorem, not a hope: you must not be able to fake a win. That's average-case hardness — and Ajtai handed us that forty years ago. Short vectors in a random lattice are as hard to find as the worst case of the whole problem. I just push his reduction through at my exact β.
